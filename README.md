# Creatives

A modern React + Vite + TypeScript website with smooth animations and a creative gallery showcase.

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **React Router** - Routing
- **@studio-freight/lenis** - Smooth scrolling

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm (recommended) or npm

### Installation

```bash
# Install dependencies
pnpm install
```

### Development

```bash
# Start development server
pnpm dev
```

### Build

```bash
# Build for production
pnpm run build
```

### Preview

```bash
# Preview production build
pnpm run preview
```

## Scripts

- `pnpm dev` - Start development server
- `pnpm build` - Build for production
- `pnpm preview` - Preview production build
- `pnpm lint` - Run ESLint
- `pnpm typecheck` - Run TypeScript type checking

## Features

- Smooth scroll animations with Lenis
- Responsive design with Tailwind CSS
- Interactive gallery component
- Custom cursor effects
- Modern UI with Framer Motion animations

## Deployment

This project is configured for deployment on Vercel (see `vercel.json`).
